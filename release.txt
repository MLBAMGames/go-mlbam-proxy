Using Windows Powershell
> $env:GOOS = "{linux/windows}"
> $env:GOARCH = "{arm/386/amd64}"
> go build

Using Debian and Caddy
- cd /usr/local/go/bin/
- go get -u github.com/mholt/caddy/caddy
- go build github.com/mholt/caddy/caddy
- go install github.com/mholt/caddy/caddy
Caddy installed : `which Caddy`
Build mlbamproxy
- env GOOS={linux/windows/darwin} GOARCH={386/amd64} go build
- e.g GOOS=windows GOARCH=386 go build
- zip windows.zip {executable}
